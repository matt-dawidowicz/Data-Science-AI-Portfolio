# Changelog

All notable changes to this educational linked-structure package are tracked
here.

## 0.1.0 - Unreleased

- Added package metadata so the project can be installed as
  `linked-structure-lab` while exposing the `linked_list` import package.
- Marked the package as typed with `py.typed` and public generic container
  aliases.
- Added linked-list, deque, sorted-list, skip-list, unrolled-list,
  multilevel-list, positional-list, self-organizing-list, and sparse-matrix
  implementations.
- Added real-world examples, benchmark scaffolding, Mermaid visualization, and
  optional property-based invariant tests.
- Added beginner-friendly documentation, complexity notes, and a release
  readiness workflow.
